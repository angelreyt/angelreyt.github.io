Daily Burn & Daily Burn 365

Daily Burn 365 is an interactive, high intensity live workout class airing daily, on demand, year round from any device. The class encourages people to work out, stay accountable and create a fun and vibrant community. The visual branding takes on vibrant colors, playful photography and fun illustrations to create a balance between feel good and work hard.
