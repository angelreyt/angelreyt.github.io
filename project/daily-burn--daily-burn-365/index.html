Daily Burn & Daily Burn 365

Daily Burn 365 is an interactive, high-intensity live workout class airing daily on demand to wherever you are. The live workout classes are intended to encourage people to work out, stay accountable and create a fun vibrant community. The visual branding takes on energetic colors juxtaposed with joyous illustrations and light-hearted welcoming images to create a work-hard and feel-good canvas.
